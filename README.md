# 紫微知道·紫曜型格 - ZATI

### Ziwei Archetype Type Indicator · 以选择看见自己

ZATI 是一款克制、可信、带有东方文化身份感的自我探索工具。

它不替你算命，也不把人格变成诊断或命运。ZATI 通过结构化行为问卷，帮助你观察自己在不同情境下更常采取的方式，再用十六种东方人格原型，把结果写成一份可以阅读、理解和继续行动的报告。

> 选择决定类型，文化原型负责表达。

## 体验

- 24 题快速测试：用约 3 分钟获得初步人格结果
- 56 题标准测试：完成更完整的行为观察与正式报告
- 四维人格轴：把结果拆成可理解的行为倾向，而不是一个神秘标签
- 十六型人格原型：每种类型都有命核、优势、逆位表现、工作方式、关系方式与成长任务
- 仪式感揭晓：将情绪与节奏集中在结果出现的那一刻
- 可恢复答题：本地保存进度，刷新或离开后可以继续
- 分享与长文报告：先交付身份，再提供证据、边界与行动建议

## 设计立场

ZATI 的视觉系统“曜序”建立在深色数字工具的秩序感之上：曜黑背景、月白正文、星金主行动，以及低干扰的东方仪式感。

我们有意避开：

- 算命式的吉凶暗示、宿命断言与玄学恐吓
- 临床诊断、能力排名或人格优劣判断
- 卡片堆叠、超大圆角、虚假加载和付费诱导
- 用复杂术语掩盖用户真正需要回答的问题

## 开始使用

环境要求：Node.js 20+、pnpm 9+。

```bash
pnpm install
pnpm dev
```

打开 [http://127.0.0.1:5173](http://127.0.0.1:5173)。

## 常用命令

```bash
pnpm dev        # 启动开发服务器
pnpm test       # 运行 Vitest 单元与组件测试
pnpm build      # 类型检查并构建生产产物
pnpm test:e2e   # 运行 Playwright 桌面与移动流程
pnpm lint       # TypeScript 项目检查
```

## 可靠性与边界

ZATI 的人格结果来自固定的四维行为问卷与可测试的计分逻辑。题目答案、轴分数、边界状态和人格代码之间的映射都在 `src/domain/` 中维护，并由 Vitest 与 Playwright 持续验证。

本项目不构成心理诊断、医疗建议或确定性的命运预测。紫微斗数、四化和人生季节属于未来方向，目前不会伪装成已经实现的能力。
